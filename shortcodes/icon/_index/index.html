  • icon

Display one of the 2 298 icons like {{% icon name="fa-address-card" size="64px" %}} {{% icon name="fa-bell" size="32px"%}} {{% icon fa-apple b %}}

It uses :

  • font-awesome 5 free icons library .more info here

This icon shortcode will display an icon in your page.

Usage

Parameter Default Description
name required name of icon (see bellow)
size none size of icon, medium, xx-small, x-small, small, large, x-large, xx-large, 11px, 2em, 20%....
style fas use fas for solid or fab for brands see here

{{%alert info%}} Tips : setting only the name as argument works too : {{<!--*icon fa-film*/-->}}` instead of `{{<!--*icon name="fa-film"*/-->}}

Type a b letter after the name to use the brand version of FontAwesome {{<!--*icon fa-facebook b*/-->}}` instead of `{{<!--*icon name="fa-film"*/-->}}

{{%/alert%}}
