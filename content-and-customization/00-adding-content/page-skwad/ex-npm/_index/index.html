Bonjour,

Une vulnérabilité présente dans les outils de gestion de dépendance de NodeJS (YARN et NPM) permet à un agent malveillant de prendre le contrôle à distance sur la compilation de nos applications.

  • La vulnérabilité corrompt les librairies utilisées sur les projets JavaScript (Angular, VueJS, NodeJS....)
  • Les versions impactées sont les suivantes :
    • YARN : 1.21 et antérieur
    • NPM : 6.13.3 et antérieur
